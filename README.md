# Disney Animation Showcase

A Three.js animation project featuring the iconic Disney logo with animated elements and dynamic effects.

## Demo

View the live demo here: [https://disney-animation-showcase.netlify.app/](https://disney-animation-showcase.netlify.app/)

### Preview


https://github.com/user-attachments/assets/f7327f83-ecfb-4701-b4b4-15db5beac0f5


## Tech

- Three.js - 3D graphics rendering
- GSAP - Animation library
- Vanilla JavaScript
- WebGL Shaders

## Local Development

1. Clone the repository
2. Open `index.html` in a modern web browser (or use a local server)
3. The animation will start automatically
